# 🐉 iDragon Adventures

An industry-level, deploy-ready **endless runner game** built using **pure HTML, CSS, and JavaScript** — no frameworks, no build tools.
The game features smooth animations, physics-based movement, scoring system, sound effects, mobile controls, and PWA support.

## ✨ Features

| Feature                | Description                                            |
| ---------------------- | ------------------------------------------------------ |
| 🌌 3D Perspective Road | Vanishing-point road with glowing rails & parallax sky |
| 🐉 Player Animation    | Running animation + wing flutter                       |
| ⬆️ Jump Physics        | Gravity arc, double jump at higher levels              |
| ⬇️ Duck                | Slide under high obstacles                             |
| ↔️ Movement            | Move forward and backward                              |
| 🚧 Obstacles           | LOW, HIGH, and FLYING obstacles                        |
| ❤️ Lives System        | 3 lives with invincibility blink                       |
| 🏆 Scoring System      | Combo multiplier scoring                               |
| 📈 Level System        | Speed increases every level                            |
| 💾 High Score          | Stored using localStorage                              |
| 💥 Particle Effects    | Dust, fire burst, floating score text                  |
| 🔊 Sound System        | Background music + sound effects                       |
| 📱 Mobile Controls     | Touch tap, swipe, and buttons                          |
| 🌐 PWA                 | Installable and works offline                          |

---

## 🕹️ Controls

| Key       | Action         |
| --------- | -------------- |
| Space / ↑ | Jump           |
| ↓         | Duck           |
| ← / A     | Move backward  |
| → / D     | Move forward   |
| P         | Pause / Resume |
| R         | Restart        |

**Mobile Controls**

* Tap → Jump
* Swipe Down → Duck
* On-screen buttons available
